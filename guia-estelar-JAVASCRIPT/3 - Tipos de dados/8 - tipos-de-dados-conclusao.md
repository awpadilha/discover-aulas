# Tipos de Dados

Conforme o ECMAscript standard temos 9 tipos de dados:

* Data types
    * Primitive / Primitive value
    * Structural
    * Structural primitive

## Primitivos

* String
* Number
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    *...

*Function

## Primitivo Estrutural / Structural Root Primitve

* null
